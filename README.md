# Idus 모의외주

## 메인 API
1. 구매 API
2. 회원가입 / 로그인 API
   - 페이스북 소셜로그인 적용
3. 홈 화면 조회 / 작품 상세 페이지 조회  API
4. 작품 검색 / 최근 검색어 및 실시간 인기 검색어 조회 API

5. 사용자별 추천 작품 목록 조회 API
     - 구매 목록, 좋아하는 작가, 즐겨찾기한 작품, 작품 구경 수, 검색 키워드, 후기, 별점 반영
6. 후기 등록 API
